User Research Study

Comprehensive user research study including interviews, usability testing, and data analysis to improve user experience. This research project aimed to understand user behaviors, identify pain points, and provide actionable insights for design improvements.

×

Research Details

Role: UX Researcher

Timeline: 8 weeks

Methods: Interviews, Surveys, Usability Testing

Research Objectives:

• Understand user motivations and goals

• Identify usability issues in current design

• Gather feedback on proposed improvements

• Validate design decisions with data

Methodology: I conducted 12 one-on-one user interviews, distributed surveys to 150+ participants, and performed task-based usability testing with 8 users. Data analysis revealed key patterns in user behavior and preferences.

Key Findings: Users experienced difficulty with navigation flow, desired more personalization options, and preferred cleaner visual hierarchy. These insights directly informed design recommendations and prioritization.
